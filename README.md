# EchoHeart 💗🎧

**母の「聞こえない」に、愛で応えたかった。**  
これは、そんな思いから始まった、オープンソースの補聴アプリです。

## 🔊 特徴
- リアルタイムマイク入力の音声増幅
- 音が見える（ピンクの玉がぽよよん）
- 声帯域に特化したイコライザー（高音・中音・低音　＋　増幅率）
- 状況に応じたモード（環境音モード、会話モード）
- Bluetoothヘッドホン対応

## 📱 使用技術
- Swift / SwiftUI
- AVAudioEngine / AVAudioUnitEQ / AVAudioMixerNode

## 🧩 今後の展望
- 録音
- リアルタイム文字起こし（目標）
- 翻訳（目標）

## 🧡 背景
本アプリは、耳が遠くなった母のために作りました。  
市販の補聴器では届かなかった「聞こえる日常」を、アプリで少しでも届けたい──
